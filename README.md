Odin Library
A simple browser-based library app built with vanilla JavaScript, HTML, and CSS. Built as part of The Odin Project curriculum.

Features

Display books in a card layout
Add new books via a dialog form
Remove books from the library
Toggle read/unread status for each book
Each book has a unique ID generated with crypto.randomUUID()

Technologies Used

HTML
CSS
JavaScript (vanilla)

Concepts Practiced

Object constructors
Prototypes and prototype methods
DOM manipulation
Event delegation
dialog element and showModal()
event.preventDefault()
Separation of data and display logic
